# Spreadsheet Maker v2
This program retrieves items, item price, and quantity from the user and creates a neat display and total count. Version 2 now includes editing capabilities.

---

## How to Run
### In a terminal

1. Install main.py and spreadsheet.py in the same directory.

2. Run the program from the directory.
```
   python main.py
```
